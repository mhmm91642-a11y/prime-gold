# Prime Gold - Premium IPTV Website

موقع "برايم جولد" هو موقع ويب فاخر متخصص في عرض اشتراكات IPTV بتصميم عصري وأنيق يجمع بين اللونين الأسود والذهبي، مستوحى من واجهات Netflix و Apple.

## المميزات
- **تصميم فاخر:** ثيم أسود وذهبي مع تأثيرات زجاجية (Glassmorphism).
- **متجاوب بالكامل:** يعمل بشكل مثالي على جميع الأجهزة (جوال، تابلت، كمبيوتر).
- **أداء عالي:** سرعة تحميل فائقة وتحسينات SEO للغة العربية.
- **حركات احترافية:** استخدام مكتبة AOS للحركات عند التمرير وتأثيرات CSS3.
- **سهولة التواصل:** أزرار مباشرة للتواصل عبر الواتساب.

## التقنيات المستخدمة
- HTML5
- CSS3 (Custom Properties, Flexbox, Grid)
- JavaScript (Vanilla JS)
- [AOS Library](https://michalsnik.github.io/aos/) (Animations)
- [Font Awesome](https://fontawesome.com/) (Icons)
- [Google Fonts](https://fonts.google.com/) (Cairo Font)
